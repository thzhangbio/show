# Tianhui Show

一个零构建的静态展示站，适合托管可以直接分享给读者的独立 HTML 文件。

## 目录约定

- `index.html`: 首页入口
- `styles.css`: 首页样式
- `works/`: 存放实际对外展示的 HTML 文件

## 发布方式

这个项目适合直接连接 GitHub 到 Vercel 部署。

## 后续更新

1. 把新的 HTML 文件放进 `works/`
2. 推送到 GitHub
3. 等待 Vercel 自动部署完成
4. 通过 `https://你的域名/works/文件名.html` 分享给读者
